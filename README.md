# Otakudesu AdBlock

Ekstensi Chrome buat nutup iklan di **otakudesu.best**.  
Blok elemen seperti:
- .iklan
- #iklantengah1
- .iklanpost
- .box_item_ads_popup
- #overplay
- #iklanbawah
- bisa kostum sendiri dah serah lu

Juga stop script dari:
- https://ads.desustream.com/ads/* <- kampret
